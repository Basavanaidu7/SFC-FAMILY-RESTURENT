# SFC Family Restaurant Frontend

React + TypeScript + Vite app for the SFC Family Restaurant customer flow.

Source folders:

- `src/pages/` - route screens.
- `src/context/` - checkout/cart provider.
- `src/lib/` - Firebase and local profile helpers.
- `src/styles/` - global CSS.

Useful commands:

```bash
npm run dev
npm run build
npm run preview
```

The backend lives in `../backend`.
